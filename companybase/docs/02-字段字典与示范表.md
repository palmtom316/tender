# 字段字典与示范表

本文件说明 `templates/companybase_master.xlsx` 各 sheet 的字段含义、系统映射和示范值。

## 1. 公司主体

| 字段 | 必填 | 系统字段 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| company_key | 是 | library_company.company_key | 稳定唯一键 | main_company |
| company_name | 是 | library_company.company_name | 公司全称 | 某某电力工程有限公司 |
| company_type | 否 | library_company.company_type | bidder/branch/partner/supplier | bidder |
| enabled | 是 | library_company.enabled | TRUE/FALSE | TRUE |
| metadata_json | 否 | library_company.metadata_json | JSON 扩展字段 | {"source":"companybase"} |

## 8. 制度能力

制度能力当前建议作为 `evidence_asset` 或模板上下文资料导入，也可先维护在本 sheet，后续导入为 `evidence_asset` 或 `company_assets.evidence_assets`。

| 字段 | 必填 | 说明 |
| --- | --- | --- |
| unique_key | 是 | `method_quality_management_system` |
| company_key | 是 | 关联公司主体 |
| capability_domain | 是 | quality/safety/green/schedule/digital/emergency/standardization |
| title | 是 | 制度或能力名称 |
| summary | 是 | 可用于投标正文的摘要 |
| applicable_chapters | 否 | 8;9;10.1;10.2;10.3 |
| evidence_file_path | 否 | 对应制度文件或证明文件 |
| metadata_json | 否 | JSON 扩展 |

## 9. 附件索引
