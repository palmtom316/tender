"""UAT checklist for the AI-assisted tender system Phase 1."""

# UAT 验收检查清单

## 主链路验收（端到端）

- [ ] 创建项目并上传招标文件
- [ ] 文档解析成功，可查看章节结构和表格
- [ ] 否决项识别并逐条人工确认
- [ ] 评分标准结构化入库
- [ ] 章节生成并编辑
- [ ] 审校发现问题并处理
- [ ] 响应矩阵覆盖状态正确
- [ ] 导出门禁三道检查通过
- [ ] DOCX 导出成功
