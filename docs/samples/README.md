# 验收样本需求（B-01）

> **截止日期：Week 1 结束（2026-03-22）**
> **状态：待落实**

## 需求清单

### 1. 真实招标文件（2-3 份）

需覆盖以下特征：

- [ ] 至少 1 份包含明确否决项条款
- [ ] 至少 1 份包含评分标准表（带分值和评分方式）
- [ ] 至少 1 份包含资质/人员/业绩要求
- [ ] 建议覆盖不同专业类型（土建、安装、市政等）

文件格式：PDF（招标文件原件扫描或电子版均可）

### 2. 期望导出样例（1 套）

- [ ] 一份完整技术标 Word 文档，作为导出模板和验收参照
- [ ] 包含章节结构、格式要求（字体、字号、行距、页边距）
- [ ] 标注哪些章节需要 AI 生成，哪些需要人工填写

## 存放位置

将样本文件放置于 `docs/samples/` 目录下，按项目编号组织：

```
docs/samples/
  sample-01/
    招标文件.pdf
    说明.md
  sample-02/
    招标文件.pdf
    说明.md
  expected-output/
    技术标模板.docx
    格式说明.md
```
