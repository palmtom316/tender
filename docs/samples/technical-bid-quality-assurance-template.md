# 九、质量保证措施

> 使用说明：本模板适用于电力工程、配网工程、用户高低压供配电工程等技术标中的“质量保证措施”章节。使用时应根据招标文件、施工图纸、技术规范书和项目实际情况替换 `{变量}`，并删除不适用的工序模块。招标文件或技术规范书另有更高要求的，以招标文件和技术规范书为准。

## 本章评分响应导读

国网及其所属单位技术标评审通常不只看“有没有质量体系”，更关注质量措施是否能落到工程现场、是否覆盖配网工程常见质量风险、是否能支撑履约评价和达标投产要求。本章按“目标可量化、责任可追溯、过程可检查、问题可闭环、资料可移交”的口径组织内容，便于评审专家快速识别质量管理能力。

| 评审关注点 | 本章响应位置 | 响应方式 |
| --- | --- | --- |
| 质量目标是否明确 | 9.1.2、9.1.3 | 将合格率、资料同步、隐蔽验收、投运验收等目标分解到责任岗位和控制措施。 |
| 质量体系是否健全 | 9.3 | 设置项目经理、技术负责人、质量负责人、材料负责人、专业负责人、资料员的职责边界。 |
| 是否符合国网标准化管理要求 | 9.2、9.3、9.4 | 引入标准化项目部、作业指导书、验收节点、档案移交、承包商评价等管理口径。 |
| 关键工序是否有实质控制 | 9.6 | 对基础、电缆、柜体、变压器、架空线路、自动化、计量、顶管等工序列出控制要点。 |
| 是否能防止质量通病 | 9.7 | 对配网工程常见质量通病列出易发部位、主要原因、防治措施和验收关注点。 |
| 送电前是否有专项把关 | 9.8 | 对安装状态、试验结果、相序回路、缺陷闭环、资料齐备性和现场状态进行联合确认。 |
| 质量问题是否闭环 | 9.9 | 按发现、分析、整改、复查、归档、复盘形成闭环。 |
| 资料是否满足验收移交 | 9.10 | 明确质量资料与实体工程同步形成、同步检查、同步移交。 |
| 是否主动协同验收 | 9.11 | 明确业主、监理、运行单位提前介入、联合检查和缺陷销项机制。 |
| 是否保障履约评价 | 9.12 | 将质量问题整改、过程留痕、班组考核和保修服务纳入履约评价保障。 |
| 是否有差异化亮点 | 9.13 | 设置二维码质量档案、样板先行、首件验收、送电前“三核三查”、质量负面清单等可落地亮点。 |
| 图表是否清楚 | 附：本章建议配套图表 | 设置质量管理体系图、流程图、矩阵表、控制点表、验收闭环图等。 |

## 9.1 编制依据与质量目标

### 9.1.1 编制依据

本章质量保证措施依据 `{项目名称}` 招标文件、技术规范书、施工图纸、答疑补遗文件、现行国家及行业标准、国家电网/南方电网等发包人相关管理要求以及我公司质量管理体系文件编制。施工过程中，如标准、规范或发包人管理要求发生更新，按最新有效版本执行；如各项要求之间存在差异或冲突，及时提请发包人、监理单位和设计单位确认，并以确认后的书面意见作为施工和验收依据。

本工程质量管理重点围绕 `{施工范围}`、`{关键设备或关键工序}`、`{工程地点及现场条件}`、`{停电或带电作业要求}`、`{工期节点}` 等内容展开，确保质量控制贯穿施工准备、材料设备进场、过程施工、隐蔽验收、试验调试、竣工验收和保修服务全过程。

### 9.1.2 质量目标

我公司承诺严格执行招标文件、合同文件、设计文件和现行验收规范要求，建立覆盖项目全过程的质量保证体系，实现以下质量目标：

1. 工程质量满足国家、行业、地方及发包人现行验收标准要求，分项、分部、单位工程验收合格率达到 100%。
2. 主要材料、设备、构配件进场验收合格率达到 100%，未经检验或检验不合格的材料、设备不得用于本工程。
3. 隐蔽工程、关键工序、停工待检点、见证点和旁站点按规定履行验收手续，验收资料真实、完整、可追溯。
4. 电气安装、土建施工、试验调试、计量装置安装等施工质量满足设计文件和相关规范要求，工程实体无永久性质量缺陷。
5. 不发生一般及以上施工质量事故，不发生因施工质量问题导致的返工、投运延期或运行隐患。
6. 工程竣工资料、试验报告、隐蔽验收记录、质量评定资料与工程实体同步形成，满足发包人档案移交要求。
7. 具备投运条件的工程，配合完成启动验收、送电投运和移交工作，确保投运过程安全、可靠、一次验收合格。

如招标文件对质量目标、达标投产、优质工程、创优目标或履约评价另有明确要求，我公司将按招标文件要求细化目标分解，并纳入项目质量计划和质量责任考核。

## 9.3 质量保证体系

### 9.3.1 质量管理组织

项目部成立以项目经理为第一责任人的质量管理组织，设置技术负责人、质量负责人、材料负责人、资料员、各专业施工负责人和班组质量员。质量管理实行“项目经理统一领导、技术负责人技术控制、质量负责人过程监督、专业负责人组织落实、班组自检互检”的管理模式。

质量管理组织建议如下：

```text
项目经理
├─ 技术负责人
│  ├─ 土建/基础施工负责人
│  ├─ 电气安装负责人
│  ├─ 电缆/线路作业负责人
│  └─ 调试试验负责人
├─ 质量负责人
│  ├─ 班组质量员
│  └─ 隐蔽验收/资料复核人员
├─ 安全负责人
├─ 材料负责人
└─ 资料员/信息员
```

正式投标文件中可将上述结构渲染为“质量管理体系机构图”。

### 9.3.3 国网标准化质量管控机制

结合国网配网工程标准化管理要求，本工程质量管理采用“标准化项目部、标准化作业、标准化验收、标准化资料”的四级控制机制。

1. **标准化项目部。** 项目部按发包人要求配置管理人员，明确项目经理、技术负责人、质量负责人、安全负责人、资料员和各专业负责人职责。开工前完成项目管理策划、质量计划、人员资质核查、施工机具和检测仪器核查，确保项目部具备独立履约和质量管控能力。
2. **标准化作业。** 对电缆敷设、终端制作、柜体安装、变压器安装、柱上设备安装、接地施工、自动化调试、计量装置安装等作业，统一采用经审批的施工方案、作业指导书和标准工艺卡。作业前交底，作业中检查，作业后验收，避免同类工程因班组差异造成质量波动。
3. **标准化验收。** 将工序验收点分为班组自检点、项目专检点、监理见证点、停工待检点和隐蔽验收点。对影响投运安全和运行可靠性的工序，实行“未验收不得隐蔽、未试验不得送电、未闭环不得移交”。
4. **标准化资料。** 施工记录、材料合格证、试验报告、隐蔽验收、影像资料、缺陷整改、竣工图和移交清单按发包人档案目录同步收集。资料内容与现场实体、设备编号、回路编号、试验数据保持一致，满足后续运行检修和履约评价要求。

## 9.4 全过程质量控制措施

### 9.4.1 施工准备阶段质量控制

1. **图纸会审与现场复核。** 开工前组织项目经理、技术负责人、质量负责人、专业施工负责人对设计图纸、技术规范书、施工范围和现场条件进行复核，重点核查设备型号、安装位置、线路路径、电缆通道、基础尺寸、接地方式、与既有设施接口等内容。发现设计与现场不一致、地下管线不明、设备安装空间不足、停电窗口受限等问题时，及时形成问题清单并提交发包人、监理单位和设计单位确认。
2. **施工方案与作业指导书。** 对 `{关键工序}`、隐蔽工程、停电作业、带电邻近作业、吊装运输、顶管/拉管、试验调试等内容编制专项施工方案或作业指导书，明确施工流程、质量标准、控制点、验收要求和应急措施。方案经审批后实施，未经审批不得擅自施工。
3. **技术质量交底。** 开工前由技术负责人向施工负责人、班组长和作业人员进行书面交底，交底内容包括设计要求、施工工艺、质量标准、允许偏差、材料设备要求、成品保护、质量通病预防、验收资料要求等。交底记录由交底人和接受交底人员签字确认。
4. **计量器具和检测设备校验。** 使用的全站仪、水准仪、兆欧表、接地电阻测试仪、耐压试验设备、扭矩工具等应在检定或校准有效期内。未经检定、超过有效期或状态异常的检测设备不得使用。
5. **样板引路和首件认可。** 对重复性强、质量观感要求高或易发生通病的工序，实施样板引路或首件认可。首件经项目部、监理单位或发包人确认后，再组织大面积施工。

### 9.4.2 材料设备进场质量控制

1. **进场资料核验。** 材料、设备进场时核查产品合格证、出厂试验报告、型式试验报告、检测报告、说明书、装箱清单、质量证明文件及招标文件要求的其他资料。
2. **外观和规格检查。** 对设备型号、规格、数量、外观、附件、备件、铭牌、包装、密封、运输损伤等进行检查。高低压柜、环网柜、箱变、变压器、电缆、导线、金具、计量表计等重点核对型号规格与设计图纸一致性。
3. **抽样复检和见证取样。** 对钢筋、水泥、砂石、混凝土试块、电缆、导线、接地材料等需复检材料，按规范和监理要求进行见证取样、送检，复检合格后方可使用。
4. **标识、存放与防护。** 材料设备按品种、规格、批次分类堆放，设置状态标识。电气设备采取防潮、防尘、防碰撞措施；电缆端部封堵严密；精密仪器、计量设备专人保管。
5. **不合格品处置。** 检验不合格、资料不齐、型号不符或存在损伤的材料设备，及时隔离、标识并办理退场或更换手续，严禁进入施工工序。

### 9.4.3 过程施工质量控制

1. **三检制度。** 严格执行班组自检、专业互检、项目专检制度。上道工序未经验收合格，不得进入下道工序；自检不合格不移交，互检不合格不接收，专检未签认不施工下一步。
2. **三工序管理。** 坚持“检查上道工序、保证本道工序、服务下道工序”，对工序交接部位、接口尺寸、设备基础、线缆端接、接地连接、隐蔽部位等进行重点复核。
3. **关键控制点管理。** 对见证点、停工待检点、旁站点和隐蔽验收点实行清单化管理。未经监理或发包人确认的停工待检点，不得擅自隐蔽或继续施工。
4. **质量通病预防。** 针对基础尺寸偏差、预埋件偏位、混凝土裂缝、接地不可靠、电缆弯曲半径不足、二次接线错误、柜体垂直度超差、相序错误、标识缺失、资料滞后等问题制定预控措施。
5. **施工记录同步。** 施工日志、测量记录、隐蔽工程验收记录、材料验收记录、试验记录、缺陷整改记录与现场施工同步形成，做到工程实体、质量记录和影像资料一致。

### 9.4.5 隐蔽工程质量控制

隐蔽工程实行“班组自检、项目复检、监理验收、资料归档”的控制程序。隐蔽前应完成尺寸复核、材料检查、施工质量检查和影像留存。隐蔽验收内容包括但不限于：基础钢筋、预埋件、接地装置、沟槽及排管、顶管/拉管、管线交叉保护、电缆敷设前通道检查、二次接线隐蔽部位等。

隐蔽工程验收记录应载明验收部位、检查项目、设计要求、实测结果、验收结论、整改情况及验收人员签字。未经监理单位验收合格或未形成验收记录的隐蔽工程，不得进行覆盖、回填、封堵或进入下道工序。

### 9.4.6 试验调试质量控制

试验调试前编制试验计划，明确试验项目、试验方法、试验设备、试验标准、人员分工和安全措施。试验设备应检定合格，试验人员应具备相应能力。电缆绝缘、电气设备交接试验、接地电阻、保护装置整定、遥信遥测遥控、计量装置功能检查等项目按规范和发包人要求执行。

试验过程中如发现数据异常、绝缘不合格、接线错误、机构动作异常、通信异常或保护逻辑不正确，应立即停止相关后续工作，查明原因并整改。整改完成后重新试验，直至满足要求。试验报告经试验负责人、技术负责人审核后归档。

### 9.4.7 竣工验收与移交质量控制

工程完工后，项目部先组织内部预验收，对实体质量、设备状态、标识标牌、接地、封堵、清洁、资料完整性进行全面检查。对发现的问题建立缺陷清单，明确责任人、整改措施和完成时限，整改完成后复查销项。

竣工验收前提交竣工资料、试验报告、隐蔽验收资料、材料设备合格证明、变更签证、影像资料、设备说明书和移交清单。对影响启动投运的问题，必须在投运前整改完成；对不影响投运但需完善的问题，按发包人和监理单位要求限期闭环。

## 9.5 质量管理制度

### 9.5.1 技术交底制度

每项分部分项工程或关键工序施工前，必须进行技术质量交底。交底应覆盖施工范围、施工方法、质量标准、允许偏差、检验方法、成品保护、质量风险和资料要求。交底记录归入项目质量资料，作为过程检查和责任追溯依据。

### 9.5.2 质量例会制度

项目部每周组织一次质量例会，必要时根据工程进展增加专题会议。会议内容包括本周质量检查情况、质量问题整改情况、下周关键质量控制点、材料设备到货验收、试验调试安排和需协调事项。会议形成纪要并跟踪闭环。

### 9.5.3 监督检查制度

项目部定期开展质量巡检和专项检查，对关键工序、隐蔽工程、设备安装、试验调试、资料同步情况进行重点检查。主动接受发包人、监理单位和运行单位检查，对检查意见建立整改台账，做到问题有记录、整改有措施、复查有结论。

### 9.5.4 工序交接制度

前后衔接工序之间必须办理交接检查。交接内容包括实体质量、尺寸偏差、预留预埋、成品保护、资料记录和遗留问题。未办理交接或交接不合格的，不得进入下一工序。

### 9.5.5 质量问题整改闭环制度

对发现的质量问题，按“问题提出、原因分析、措施制定、整改实施、复查确认、资料归档”的流程闭环。对重复发生或影响较大的质量问题，组织技术负责人、质量负责人和施工班组进行专题分析，采取工艺优化、人员培训、材料更换或方案调整等措施，防止再次发生。

### 9.5.6 质量奖惩制度

项目部将质量目标、过程检查结果、整改闭环情况与班组考核挂钩。对严格执行工艺标准、质量记录完整、一次验收合格率高的班组和个人予以奖励；对违反工艺标准、造成返工或质量隐患的，按项目管理制度追责并落实整改。

### 9.6.0 质量管控亮点措施

为体现我公司对国网配网工程质量管理要求的理解，本工程在常规质量控制基础上，进一步采取以下亮点措施：

1. **一工序一清单。** 对基础施工、电缆敷设、柜体安装、接地施工、二次接线、试验调试等关键工序分别编制质量检查清单，作业人员按清单施工，质量负责人按清单检查，资料员按清单归档。
2. **一设备一档案。** 对变压器、环网柜、箱变、高低压柜、柱上开关、配网自动化终端、计量装置等主要设备建立设备质量档案，档案内容包括到货验收、安装检查、试验数据、缺陷整改、投运确认和移交资料。
3. **一缺陷一闭环。** 对检查发现的质量缺陷建立台账，明确缺陷描述、责任人、整改措施、完成时限、复查结论和影像资料，做到缺陷不闭环不验收、不移交。
4. **一节点一确认。** 对开工、材料进场、隐蔽工程、关键安装、试验调试、送电前检查、竣工移交等节点实行签认制度，避免施工进度推进快于质量确认。
5. **一现场一复盘。** 对质量问题、返工事项、监理检查意见和运行单位验收意见进行复盘，形成同类项目质量风险提示，反向完善作业指导书和班组交底内容。

### 9.6.2 电缆线路质量控制

1. 电缆到货后核对型号、规格、电压等级、长度、合格证和试验报告，检查外护套、端部封堵和盘号标识。
2. 敷设前检查电缆通道、排管、沟槽、桥架、转弯半径、牵引路径和障碍物，确认具备敷设条件。
3. 敷设过程中控制牵引力、侧压力和弯曲半径，防止电缆外护套损伤、扭曲、过度弯折和交叉混乱。
4. 电缆排列整齐、固定牢靠、标识清晰，首端、末端、转弯处、接头处和穿越部位按要求设置标牌。
5. 电缆终端和接头制作由合格人员按工艺要求实施，制作环境保持清洁干燥，完成后进行外观检查和电气试验。
6. 电缆敷设、终端接头制作、试验和隐蔽封堵形成完整记录。

### 9.6.4 变压器安装质量控制

1. 变压器进场后检查本体、附件、油位、密封、瓷套、冷却装置、铭牌、出厂资料和试验报告。
2. 二次搬运和吊装由具备经验的人员组织实施，索具、吊点、牵引点和运输路线经检查确认，防止瓷套、油箱和附件损伤。
3. 就位前核对高低压侧方向、基础尺寸、轨道或底座位置、距墙和检修空间，安装后检查水平度、固定和防震措施。
4. 一、二次引线连接不得使套管直接承受外力，中性点、外壳和附件接地可靠，接地连接可检查、可维护。
5. 安装完成后按交接试验要求进行绝缘、电阻、变比、极性、耐压、接地等试验，试验合格后方可进入后续投运程序。

### 9.6.5 架空线路及柱上设备质量控制

1. 杆塔基础、杆位、拉线、横担、金具、绝缘子、导线型号和相序按设计要求施工。
2. 导线展放、紧线、弧垂观测和附件安装按规范执行，弧垂误差、相间距离、对地距离和交叉跨越距离满足要求。
3. 裸导线与线夹、绝缘子接触处按要求采取保护措施；绝缘导线固定和连接不得损伤绝缘层。
4. 柱上变压器、柱上开关、隔离开关、跌落式熔断器、避雷器等安装牢固、接线可靠、标识清晰、接地可靠，操作机构灵活。
5. 完成后进行外观检查、接地检查、相序核对和必要试验，确认满足送电条件。

### 9.6.6 配网自动化及计量装置质量控制

1. 配网自动化终端、通信设备、控制电缆、电源装置、电流电压回路设备的型号、规格、配置满足设计和技术规范书要求。
2. 控制电缆敷设路径、屏蔽接地、端子编号、芯线标识、回路接线和绝缘测试符合要求，电流回路严禁开路，电压回路严禁短路。
3. 计量箱、电能表、采集终端安装位置便于运行维护，固定牢靠，接线正确，封印、标识和调试记录完整。
4. 自动化设备安装完成后进行通电检查、通信测试、遥信遥测遥控核对、保护或控制逻辑检查，确认数据上传和现场状态一致。

### 9.6.7 顶管、拉管及地下通道质量控制

如本工程涉及顶管、拉管、排管或电缆通道穿越，应重点控制以下内容：

1. 施工前查明地下管线、构筑物、道路、河道、既有电缆和其他障碍物，必要时开挖探槽并形成管线调查记录。
2. 复核出入土点、轴线、标高、转弯半径、工作坑位置和保护范围，确认与设计一致。
3. 顶管或拉管设备进场前完成检查和调试，操作人员持证上岗，泥浆配比、顶进速度、顶力、回拖力等参数按方案控制。
4. 施工过程中坚持“勤测量、勤纠偏、小量纠”的原则，及时记录轴线、高程、顶力、泥浆、沉降观测等数据。
5. 穿越道路、重要构筑物或既有管线时，设置沉降观测点和预警值，按方案规定频率观测，发现异常立即停工分析并采取加固、调整参数或回填注浆等措施。
6. 完成后检查管道通畅性、接口质量、封堵、防水、防沉降和回填密实度，形成验收记录。

## 9.7 质量通病专项防治措施

| 序号 | 薄弱环节 | 可能影响 | 预控措施 |
| --- | --- | --- | --- |
| 1 | 施工人员质量意识不足 | 造成工艺粗糙、返工、隐患遗留 | 开工前质量培训，班前交底，关键岗位安排经验人员，质量奖惩与班组考核挂钩。 |

| 5 | 电缆敷设和接头制作质量不稳定 | 绝缘下降、运行故障 | 控制牵引力和弯曲半径，接头制作环境清洁干燥，完成后进行试验。 |

针对配网工程常见质量通病，项目部建立专项防治清单，施工前在技术交底中明确预防措施，施工中由质量负责人跟踪检查，验收时逐项复核。对重复出现的问题，纳入班组质量考核和项目复盘。

## 9.8 送电前质量专项检查

送电前质量检查是本工程质量控制的最后一道关键关口。项目部在申请送电或启动投运前，组织项目经理、技术负责人、质量负责人、调试试验负责人、专业施工负责人和资料员开展专项检查，并邀请监理单位、发包人代表或运行单位参加联合确认。

送电前专项检查包括以下内容：

1. **设备安装状态检查。** 核查变压器、柜体、环网柜、柱上设备、计量装置、自动化终端等安装位置、固定、接地、封堵、标识、相色和外观状态，确认无影响投运的实体缺陷。
2. **电气试验结果检查。** 核查交接试验、绝缘测试、接地电阻、主回路电阻、保护调试、遥信遥测遥控、计量采集通信等试验项目是否齐全，试验结论是否合格。
3. **相序、极性和回路检查。** 对一次相序、二次回路、保护极性、计量接线、自动化信号、通信链路进行复核，避免送电后出现保护误动、计量异常或自动化数据错误。
4. **缺陷整改闭环检查。** 对施工过程、监理检查、发包人检查、运行单位预验收提出的问题逐项销号，影响投运安全和运行可靠性的缺陷未闭环前不得送电。
5. **资料齐备性检查。** 核查试验报告、隐蔽验收、材料设备资料、安装记录、调试记录、缺陷销项表、竣工图和移交清单，确保资料能支撑送电确认和后续运维。
6. **现场状态检查。** 检查临时接线、施工工具、杂物、孔洞封堵、安全围栏、警示标识、设备命名牌和运行标识，确认现场具备安全投运条件。

送电前质量专项检查采用清单化管理，检查结论由责任人签字确认。检查发现的问题按缺陷等级分类处理，影响送电安全的问题立即整改并复查；不影响送电但需完善的问题列入遗留问题清单，明确责任人和完成时限，经发包人或运行单位认可后闭环。

## 9.9 质量问题处置和持续改进

施工过程中发现质量问题时，项目部立即组织分析，区分一般质量缺陷、重复性质量问题和影响投运的重大质量隐患，分别制定处置措施。对一般缺陷及时整改并复查；对重复发生的问题，组织技术负责人、质量负责人和班组进行原因分析，修订作业指导书或交底内容；对可能影响安全运行、投运节点或验收结论的问题，立即上报发包人和监理单位，按批准方案处理。

项目部对质量问题实行闭环管理，整改完成后由质量负责人复查确认，必要时邀请监理单位复验。所有问题处理过程形成记录，纳入竣工资料和项目复盘资料。工程结束后，总结质量管理经验和质量通病防治效果，形成可复用的质量控制清单，用于后续同类工程持续改进。

## 9.10 质量资料同步管理措施

质量资料管理坚持“真实、完整、同步、可追溯”的原则。资料员根据发包人档案要求建立资料目录，施工过程中同步收集和整理以下资料：

1. 开工资料、施工组织设计、专项方案、技术交底记录。
2. 图纸会审、设计变更、工程联系单、签证资料。
3. 材料设备合格证、出厂试验报告、进场验收记录、复检报告。
4. 测量记录、隐蔽工程验收记录、工序交接记录、质量检查记录。
5. 电气试验报告、调试记录、保护整定记录、通信联调记录。
6. 质量问题整改记录、监理通知回复、复查记录。
7. 竣工图、竣工验收资料、设备移交清单、运行维护资料。

资料整理应做到分类清晰、编号连续、签字盖章齐全、结论明确。电子资料与纸质资料保持一致，影像资料按施工部位、日期和工序命名归档。

为满足国网项目对过程留痕和档案移交的要求，项目部实行“五同步”资料管理：

1. **一工序一资料。** 每项关键工序完成后同步形成施工记录、检查记录和验收记录，不在竣工阶段集中补填。
2. **一设备一档案。** 对主要设备建立从开箱验收、安装、试验、缺陷处理到投运确认的完整质量档案。
3. **一隐蔽一影像。** 隐蔽工程验收前留存关键部位影像资料，影像编号与隐蔽验收记录、施工部位保持一致。
4. **一缺陷一闭环。** 每项质量缺陷均形成问题台账、整改措施、复查结论和销项记录。
5. **一验收一签认。** 材料进场、隐蔽验收、关键工序、试验调试、送电前检查、竣工移交均形成签认资料。

| 到货验收 | 合格证、出厂试验报告、装箱清单、开箱记录 | 材料负责人 | 型号规格、附件备件、外观状态 |
| 安装施工 | 安装记录、接地记录、二次接线记录、影像资料 | 专业负责人 | 安装质量、接地、封堵、标识 |
| 试验调试 | 交接试验报告、保护调试记录、通信联调记录 | 调试试验负责人 | 试验项目、数据、结论 |
| 缺陷整改 | 缺陷台账、整改记录、复查记录 | 质量负责人 | 缺陷是否闭环，是否影响投运 |
| 投运移交 | 送电前确认表、运行移交清单、设备资料 | 资料员 | 资料完整性、图实一致性 |

## 9.11 业主、监理、运行单位协同验收机制

本工程质量验收坚持施工单位自控与外部监督相结合。项目部主动接受发包人、监理单位和运行单位的质量监督，将关键节点报验、联合检查和缺陷销项纳入项目质量计划。

1. **关键节点提前报验。** 对材料设备进场、隐蔽工程、关键设备安装、电气试验、送电前检查等节点，提前向监理单位和发包人报验，确保相关单位有充足时间安排见证或检查。
2. **隐蔽工程共同验收。** 对接地、基础钢筋、预埋件、排管、沟槽、电缆敷设等隐蔽工程，按规定完成自检和复检后报监理验收，必要时邀请运行单位或发包人参加。
3. **运行单位提前介入。** 对涉及后续运行维护的设备命名、标识、通道、接地、自动化信息、计量采集等内容，主动征求运行单位意见，减少投运移交阶段整改。
4. **送电前联合检查。** 送电前由项目部组织联合检查，对设备状态、试验报告、缺陷闭环、资料齐备性和现场安全状态进行确认。
5. **缺陷清单销项。** 对发包人、监理单位、运行单位提出的问题建立统一销项台账，明确责任人和完成时限，整改完成后提交复查确认。

协同验收流程如下：施工单位自检合格 -> 项目部复检 -> 监理单位验收 -> 发包人/运行单位抽查或联合检查 -> 问题整改 -> 复查销项 -> 资料归档 -> 进入下一节点。

## 9.12 质量履约评价保障措施

国网项目质量管理结果会直接影响承包商履约评价和后续市场信誉。项目部将质量履约评价要求前置到施工全过程，确保工程实体、过程记录、整改闭环和移交资料均满足发包人评价要求。

1. **减少质量类扣分事项。** 对监理通知、发包人检查意见、运行单位验收意见及时响应，避免因重复性质量问题、整改不及时、资料缺失、验收不配合等事项影响履约评价。
2. **加强过程留痕。** 对关键工序、隐蔽工程、试验调试、送电前检查和缺陷整改保留完整记录，为履约评价、竣工结算和后续运维提供依据。
3. **班组质量考核。** 将质量检查结果、返工次数、缺陷整改及时性和资料完整性纳入班组考核，促使一线人员重视施工质量。
4. **质量复盘提升。** 工程结束后对质量目标完成情况、质量问题、整改效率、验收意见和资料移交情况进行复盘，形成同类项目质量管理改进清单。
5. **服务延伸。** 投运移交后继续配合发包人和运行单位处理质量保修、资料补充和运行反馈问题，维护项目履约评价和企业信誉。

## 9.13 质量管理创新与亮点措施

为进一步提高本工程质量管理的可视化、可追溯和可验收水平，我公司在常规质量保证措施基础上，结合国网配网工程标准化管理和运行移交要求，设置以下差异化亮点措施。相关措施均以现场可执行、资料可留痕、验收可核查为原则，不增加发包人管理负担。

### 9.13.1 一设备一二维码质量档案

对 `{关键设备或关键工序}` 中涉及的箱变、环网柜、高低压柜、变压器、柱上开关、电缆终端、计量装置、配网自动化终端等主要设备，建立“一设备一档案”。具备条件时，将设备编号、安装位置、合格证、出厂试验报告、开箱验收记录、安装检查记录、交接试验报告、缺陷整改记录、送电前确认记录和移交资料进行二维码关联，便于发包人、监理单位和运行单位在现场快速核查。

二维码质量档案不替代正式竣工资料，而作为现场核验和移交辅助工具。每台设备的质量档案应至少包含以下内容：

| 档案类别 | 内容 | 质量价值 |
| --- | --- | --- |
| 设备基础信息 | 设备名称、编号、型号、安装位置、生产厂家 | 便于现场设备与资料快速对应 |
| 到货验收资料 | 合格证、出厂报告、装箱清单、开箱记录 | 证明设备来源清楚、资料完整 |
| 安装过程资料 | 安装记录、接地记录、封堵检查、影像资料 | 证明安装过程可追溯 |
| 试验调试资料 | 绝缘、耐压、接地、保护、遥信遥测遥控、计量采集等记录 | 支撑送电和投运条件确认 |
| 缺陷闭环资料 | 缺陷清单、整改记录、复查结论 | 证明问题已处理并销项 |
| 移交资料 | 送电前确认表、运行移交清单、运维注意事项 | 便于运行单位接收和后续维护 |

### 9.13.2 样板先行与首件验收

对重复性强、质量观感要求高、运行风险高的作业，实行“样板先行、首件验收、批量推广”。首件样板经项目部、监理单位或发包人确认后，作为后续同类工序施工和验收的参照标准。

适用样板包括但不限于：电缆终端制作、柜体接地和封堵、二次接线排布、计量箱安装、接地焊接与防腐、柱上设备引线制作、设备命名标识、竣工资料样表等。

| 样板对象 | 确认重点 | 推广要求 |
| --- | --- | --- |
| 电缆终端制作样板 | 剥切尺寸、压接、密封、相色、接地 | 样板确认后同批次终端按统一工艺制作 |
| 二次接线样板 | 线号、端子、走线、绑扎、备用芯处理 | 统一接线工艺和编号规则 |
| 柜体封堵样板 | 孔洞封堵、防小动物、防潮、防火材料 | 同类柜体封堵按样板验收 |
| 接地施工样板 | 搭接长度、焊接质量、防腐、标识 | 隐蔽前按样板标准检查 |
| 资料样板 | 表格格式、编号规则、签字盖章、影像命名 | 后续资料按样板同步形成 |

### 9.13.3 送电前“三核三查”专项把关

在送电前质量专项检查基础上，进一步执行“三核三查”机制，提升送电前最后一道质量关口的针对性。

**三核：**

1. **核相序。** 核对一次设备、电缆、计量装置、保护装置和运行方式涉及的相序，避免投运后出现相序错误。
2. **核回路。** 核对二次控制、保护、计量、自动化、通信回路，确保端子接线、线号、回路功能和图纸一致。
3. **核信号。** 核对保护信号、遥信、遥测、遥控、计量采集和通信链路，确保现场状态与后台或采集系统显示一致。

**三查：**

1. **查试验报告。** 检查交接试验、绝缘、耐压、接地、保护调试、自动化联调等报告是否齐全且结论合格。
2. **查缺陷闭环。** 检查施工、监理、发包人和运行单位提出的问题是否完成整改并复查销项。
3. **查现场状态。** 检查临时接线、施工工具、孔洞封堵、设备标识、安全围栏、警示标志和现场清洁，确认具备送电条件。

### 9.13.4 质量通病负面清单

为避免常见质量问题反复出现，项目部建立质量通病负面清单。负面清单内容在班前会、技术交底、过程检查和验收阶段反复提示，发现清单内问题时优先整改、重点复查。

| 禁止或重点避免事项 | 可能后果 | 预防要求 |
| --- | --- | --- |
| 电缆首末端、转弯处、分支处无标识或标识不清 | 后续运维识别困难，资料与现场不一致 | 敷设后立即挂牌，资料员同步更新电缆清册 |
| 二次接线线号不清、端子错接、备用芯处理混乱 | 调试失败，保护、计量或自动化异常 | 接线前核图，接线后逐芯校线，双人复核 |
| 柜体、箱变、环网柜封堵不到位 | 潮气、小动物进入，影响设备运行 | 安装完成后专项检查封堵和防潮措施 |
| 接地连接不可见、不可测或连接不牢 | 影响设备安全和验收结论 | 接地点明显可见，隐蔽前验收，完成后测试 |
| 试验报告设备编号与现场设备不一致 | 资料无法支撑投运和移交 | 试验前核对设备编号，报告与台账统一 |
| 竣工图、电缆清册、设备台账与现场不一致 | 影响运行接收和后续检修 | 施工变更即时记录，移交前专项核对 |
| 缺陷未闭环即申请送电或移交 | 形成运行隐患或履约扣分 | 缺陷销项表签认完成后进入下一节点 |

### 9.13.5 运行单位提前介入与移交前预验收

对涉及后续运行维护的内容，项目部主动邀请运行单位提前介入，重点核查设备命名、标识挂牌、运行通道、接地引下、自动化信号、计量采集、资料目录和备品备件等内容。通过移交前预验收，将运行单位可能关注的问题前置处理，减少投运移交阶段集中整改。

运行单位提前介入重点包括：

1. 设备命名和编号是否符合运行单位管理习惯。
2. 电缆标识、柜内标识、相色标识、警示标识是否清晰耐久。
3. 自动化、计量采集、通信信号是否与后台或采集系统一致。
4. 运行检修通道、设备操作空间和安全距离是否满足要求。
5. 竣工资料、设备资料、试验报告和移交清单是否便于接收。

## 9.14 质量保修承诺

我公司承诺，工程竣工验收合格并交付发包人使用后，在合同约定和法律规定的质量保修期内承担质量保修责任。保修期自竣工验收合格之日起计算，具体保修范围和期限按招标文件、合同约定及国家有关规定执行。

保修期内，如因我公司施工质量原因出现缺陷或故障，我公司在接到发包人通知后及时响应，安排专业人员到场检查，提出维修方案并实施修复。对影响安全运行或正常使用的问题，优先组织处理；对一般质量缺陷，按发包人要求限期完成整改。维修完成后形成维修记录，经发包人或运行维护单位确认后归档。

我公司将建立项目保修服务台账，记录缺陷内容、响应时间、处理措施、完成时间和确认结果。对重复性问题进行原因分析并采取预防措施，确保工程质量和运行可靠性持续满足发包人要求。
